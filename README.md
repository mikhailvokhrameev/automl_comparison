# AutoML Comparison: Heart Disease Prediction

Данный репозиторий содержит сравнительный анализ производительности библиотек AutoML на задаче предсказания сердечных заболеваний.

---

### Цель работы

Сравнить качество моделей, время обучения и простоту использования различных инструментов автоматизированного машинного обучения, а также получить практические навыки по работе с AutoML.

---

### Задача
- **Тип:** Бинарная классификация
- **Целевая переменная:** HeartDisease (наличие сердечного заболевания)
- **Метрика оценки:** F1-Score
- **Размер датасета:** 918 записей, 11 признаков

---
​
### Признаки датасета

- **Age** - возраст пациента
- **Sex** - пол (M/F)
- **ChestPainType** - тип боли в груди
- **RestingBP** - давление в покое
- **Cholesterol** - уровень холестерина
- **FastingBS** - сахар в крови натощак
- **RestingECG** - результаты ЭКГ в покое
- **MaxHR** — максимальная частота сердечных сокращений
- **ExerciseAngina** - стенокардия при нагрузке
- **Oldpeak** - депрессия ST-сегмента
- **ST_Slope** - наклон пикового сегмента ST

---

### Сравниваемые библиотеки

- **AutoGluon** — библиотека от Amazon для автоматического обучения моделей с поддержкой ансамблирования и стекинга
- **FEDOT** — фреймворк для автоматического построения композитных пайплайнов с использованием эволюционных алгоритмов
- **LightAutoML** — библиотека от Сбера с акцентом на скорость и качество
- **MLJar** — AutoML-решение с пошаговой оптимизацией и интерпретируемостью

---

### Результаты

| Библиотека  | F1-Score |
| ----------- | -------- |
| LightAutoML | 0.9135   |
| FEDOT       | 0.9100   |
| AutoGluon   | 0.9082   |
| MLJar       | 0.9082   |

---

### Вывод

Судя по таблице победил LightAutoML с F1-Score = 0.9135. В целом, все библиотеки AutoML показали очень высокий и схожий результат. Хочется выделить LightAutoML за его классную документацию и курсы на русском языке и MLJar за веб приложение и авто отчеты.
